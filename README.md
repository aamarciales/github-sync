# Github Sync

This public repository is a small static fixture for validating the Framer GitHub Releases plugin.

It is intentionally simple: one HTML page, one stylesheet, a changelog, and release note files that are used as the body for GitHub Releases.

## What This Tests

- Release titles and tags
- Markdown release notes
- Published dates
- Latest and prerelease status
- GitHub release links
- Compare links between tags
- Slugs generated from release metadata
- Author details from GitHub

## Local Preview

Open `index.html` in a browser.
